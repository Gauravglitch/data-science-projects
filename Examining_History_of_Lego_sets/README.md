# Examining the History of Lego Sets

![star_wars_unsplash](media/star_wars_unsplash.jpg)

## Overview

This project analyzes the impact of licensed Lego sets, particularly the Star Wars series, on Lego's product line and market success. We explore how partnerships with franchises like Star Wars, Super Heroes, and Harry Potter shaped Lego's growth.

## Datasets

### lego_sets.csv
Contains individual Lego set information:
- `set_num`: Unique set identifier
- `name`: Set name
- `year`: Release year
- `num_parts`: Number of pieces
- `theme_name`: Sub-theme classification
- `parent_theme`: Parent theme category

### parent_themes.csv
Contains parent theme metadata:
- `id`: Unique theme identifier
- `name`: Theme name
- `is_licensed`: Boolean indicating if theme is licensed

## Objective

Analyze the success of licensed Lego sets and their influence on the company's product strategy and growth.

## Files

- `lego_sets.csv` - Individual set records
- `parent_themes.csv` - Theme classifications
- `analysis.ipynb` - Main analysis notebook
