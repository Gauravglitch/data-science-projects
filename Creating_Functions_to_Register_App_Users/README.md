
# Creating Functions to Register App Users

## Overview
This project implements a user registration system for a mobile app that validates user input before account creation. It leverages helper functions to ensure data quality and security.

## Features
- **Name Validation**: Validates user names using the `validate_name()` function
- **Email Validation**: Checks email format with `validate_email()`
- **Password Validation**: Ensures strong passwords using `validate_password()`
- **Error Handling**: Comprehensive error handling for invalid inputs
- **User Storage**: Registers and stores validated user data

## Getting Started

### Prerequisites
- Python 3.x

### Usage
The system provides three helper functions for input validation:
- `validate_name()` - Validates user names
- `validate_email()` - Validates email addresses
- `validate_password()` - Validates password strength

Use these functions to register users securely and store their information.
